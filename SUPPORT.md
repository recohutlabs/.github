# Support

Use the affected repository's issue tracker for reproducible bugs, documentation gaps, and focused feature proposals. Include environment details, exact commands, relevant logs, and a minimal reproduction when possible.

Recohut Labs projects are experimental unless their README states otherwise. Maintainers cannot guarantee response times, implementation of feature requests, or suitability for production use.

For security concerns, follow [`SECURITY.md`](SECURITY.md) instead of opening a public issue.
